# Negocio Digital — Metodología Completa

## Tu sistema para digitalizar negocios y vender infoproductos

Este proyecto contiene TODO lo que necesitas para lanzar y vender tu metodología de digitalización de negocios. Cada carpeta es una pieza del sistema.

---

## ESTRUCTURA DEL PROYECTO

```
negocio-digital-metodologia/
│
├── 01-lead-magnet/          → Guía gratuita para captar leads
├── 02-landing-registro/     → Página para registrarse al webinar
├── 03-webinar/              → Guion completo del webinar de 45 min
├── 04-pagina-ventas/        → Página de ventas del curso principal
├── 05-emails/               → Secuencia de 7 emails automáticos
├── 06-whatsapp/             → Mensajes automáticos de WhatsApp
├── 07-curso-modulos/        → Contenido de los 8 módulos del curso
└── 08-anuncios/             → Textos para anuncios en Meta Ads
```

---

## ORDEN PARA LANZAR (paso a paso)

1. Lee la carpeta `07-curso-modulos` — entiende qué vas a enseñar
2. Crea el lead magnet de `01-lead-magnet` en Canva
3. Sube la landing de `02-landing-registro` a Systeme.io
4. Graba el webinar con el guion de `03-webinar`
5. Sube la página de ventas de `04-pagina-ventas` a Systeme.io
6. Configura los emails de `05-emails` en Systeme.io
7. Configura WhatsApp con los mensajes de `06-whatsapp` en ManyChat
8. Lanza tus primeros anuncios con los textos de `08-anuncios`
